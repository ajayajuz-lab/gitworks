>git init [for initializing git repository]

>git status [for listing untracked files]

>git add -A [adding all untracked files to staging area]

>git commit -m "message" [staged files => local repository]

>git clone [exact copy from remote repository to local workspace]

>git pull [changes from remote repository to local workspace]

>git push [local repository changes to remote repository]

>git config --list [configuration list]

>git config --global user.name [to add username]

>git config --global user.email [to add user email]

>git remote add remote_name remote_path [Add a remote repository]

>git push remote_name branch_name [To push changes from local repository to remote repository]
